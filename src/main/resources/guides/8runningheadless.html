Installing Fiji

  • Linux
  • Windows 
wget https://downloads.imagej.net/fiji/latest/fiji-linux64.zip
unzip fiji-linux64.zip
rm fiji-linux64.zip
cd Fiji.app/
[WINDOWS CODE]

Installing MIA

  • Linux
  • Windows 
./ImageJ-linux64 --update add-update-site MIA https://sites.imagej.net/ModularImageAnalysis/
./ImageJ-linux64 --update add-update-site IJPB-plugins https://sites.imagej.net/IJPB-plugins
./ImageJ-linux64 --update update
[WINDOWS CODE]

Running a workflow

  • Linux
  • Windows 
[LINUX CODE]
.\ImageJ-win64 --headless --console --run "MIA (headless)" "workflowPath='C:\\Users'"
